# サポート - YOSH!

## 不具合の報告・機能のリクエスト

不具合の報告や新機能のご要望は、以下のGitHub Issuesにてお受けしております。

https://github.com/ys39/yosh-support/issues

報告の際は、以下の情報を含めていただけるとスムーズに対応できます。

- お使いのWindowsのバージョン（例: Windows 11 24H2）
- 発生した問題の具体的な内容
- 再現手順（わかる範囲で）

## よくあるご質問

### データはどこに保存されますか？

すべてのデータは、お使いの端末内の以下のパスにSQLiteデータベースとしてローカル保存されます。インターネットへの送信は行いません。

`%APPDATA%\yosh\todo.db`

### データのバックアップ方法を教えてください。

上記パスにある `todo.db` ファイルを、任意の安全な場所にコピーしてください。復元する場合は、コピーしたファイルを同じパスに戻してからアプリを再起動します。

### アプリを初期化するにはどうすればよいですか？

以下の2つの方法があります。

- **アプリ内から:** 「設定 > データ管理」で、すべてのタスクの削除やすべてのボードの削除ができます。
- **手動で:** `todo.db` ファイルを削除してからアプリを再起動すると、すべてのデータがリセットされます。

### ボードはいくつまで作成できますか？

最大20個まで作成できます。各ボードはタブとして表示され、ドラッグ&ドロップで並び替えが可能です。

### 操作を誤ってしまいました。元に戻せますか？

タスクの削除やドラッグ&ドロップによるステータス変更の直後に、画面左下に表示されるトーストの「元に戻す」ボタンを押すと復元できます。トーストは約4秒間表示されますので、その間に操作してください。

### ダークモードに切り替えるにはどうすればよいですか？

以下のいずれかの方法で切り替えできます。

- サイドバー下部のテーマ切り替えアイコンをクリック
- 「設定 > 表示」からテーマ（ライト / ダーク / システム連動）を選択
